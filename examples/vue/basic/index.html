Basic

Explanation#

While Vue does have an official router, this project shows how you could use Curi as the router for a Vue project instead. It uses <curi-link> component provided by the the @curi/vue package.

Live Demo#

Use the three buttons at the top of the Sandbox to toggle view modes. Clicking the menu button in the top left corner opens a menu to switch between files.

On GitHub#

If you want to run this code locally, the source code is available on GitHub here.
